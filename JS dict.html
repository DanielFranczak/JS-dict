Zmienne
W Javascript zmienne nie są twardo deklarowane, zatem nie deklarujemy int, string, double etc. tylko dajemy var (starsza wersja), aktualnie wykorzystuje się let oraz const.
W tym momencie warto zwrócić uwagę na słowo hoisting Czym jest hoisting?


Funkcje
Funkcja jest to odpowiednik metody, czyli pozwala na wykonanie danej czynności po jej wywołaniu. Budowa funkcji:

function functionName(x, y) {
Calculations (e.g. var result = x + y;)
return result;
}

Wywołanie funkcji:
alert(functionName(x, y));


Instrukcje warunkowe
Instrukcje warunkowe służą do wyboru decyzji na podstawi warunku. Wyróżnia się instrukcje warunkową if, której składnia wygląda następująco:

if (warunek) {
instrukcja jeśli warunek to prawda;}
else{
instrukcja jeśli warunek to fałsz; }
Przykład:
let price = 180;
if (price > 200) {
alert("Dostales rabat")}
else {
alert("nie masz rabatu")};

Kolejny przykład to wykorzystanie bloku if else:
let price = 180;
if (price > 200) {
alert("Dostales rabat")}
if else ((price <= 200) && (price > 0)) {
alert("nie masz rabatu")}
else (alert("Blad systemu")

Pętle
Pętla while:
Przykład:
while(warunek){
instrukcja }
W przypadku pętli należy pamiętać o iteracjach, jeśli o nich zapomnimy to może powstać pętla nieskoczona. Przykład

var i = 0;
while(i < 10){
alert(i); i++}

Zatem na ekranie będą pojawiać się kolejne liczby 0,1,3...,9. Kolejną z pętli jest pętla for, działa podobnie jak while ale ma inną budowę. Budowa pętli for

for(inicjalizacja zmiennej; warunek; iteracja){
instrukcje do wykonania}

A teraz pora na przykład:

for(let i = 10 ; i > 0; i--){
alert(i)}

Taka funkcja wyświetli po kolei liczby od 10, 9, 8...,1. Istnieje również funkcja do while

Składnia takiej instrukcji to:

do{instrukcja} while (warunek); Działa ona wówczas gdy chcemy mieć pewność, że dana instrukcja została wywołana chociaż jeden raz.

Tablice
Kolejnym podstawowym elementem są tablice. Najprościej mówiąc jest to element, który zawiera inne elementy(zawiera zbiór pewnych wartości). Przykład:
var numbers = [1,23,45,11,87]; Należy pamiętać, że aby odwołać się do konkretnej wartości, należy wybrać odpowiedni index dla wartości, indexowanie zaczyna się od liczby 0. W tablicy mogą znajdować się wartości różnych typów int-y, float-y, stringi, tablice oraz inne. Przykład

var myArray = [1,2,3,"4","abc", "test"];
alert(myArray[0]);
alert(myArray[5]);

Po zastosowaniu tych instrukcji na ekranie pojawią się wybrane wartości spod numerów indexów 0 i 5, zatem będą to wartości 1 oraz test. Kolejny przykład tablicy i wyświetlania jej wartości.

var myArray = [1,2,3,"4","abc", "test"];
for(var i = 0; i < myArray.length; i++){
alert(myArray[i]);}
Wykorzystano tutaj właściwość length, która pozwala na określenie ilości wartości w tablicy. Powyższy przykład obrazuje jak w prosty sposób wypisać wszystkie wartości znajdujące się w tablicy.

Obiekty

  • Obiekty opisują realnie świat;
  • Obiekty mają swoje właściwości;
  • Składa się z pary klucz: wartość (key: value)(wartość może być funkcją!);
  • Przykład obiektu:
    var person = {
    name: 'Jan Kowalski',
    age:34,
    email:'jankowalski@gmail.com'
    sayHello: function(){
    alert('witaj' + this.name);}
    }
  • Do właściwości obiektu odwołujemy się za pomocą kropki (.) Przykład alert(person.name); Wyświetli on imię wywołanego obiektu.
  • Istnieją obiekty wbudowane do, których należą: Number, Math, String, Date, RegExp. Przykład:
    var today = new Date ();
    alert(today.getMonth));
    Taki przykład przypisuje do zmiennej aktualną datę, a następnie za pomocą funkcji alert i właściwości getMonth wyświetla aktualny miesiąc.
